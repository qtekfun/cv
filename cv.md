## Proyectos / Homelab

### `homelab-2026` — Self-hosted Infrastructure Stack
`github.com/qtekfun · qtekfun.com`

Stack de ~35 servicios Docker sobre un host **Proxmox**, con una Ubuntu VM como nodo principal.
Expuesto bajo dominio propio (`qtekfun.net`) con Cloudflare DNS. Gestionado como mono-repo
en GitHub con CI/CD propio, con planes de migración a Forgejo self-hosted.

| Capa                  | Servicios                                                                         |
|-----------------------|-----------------------------------------------------------------------------------|
| Reverse proxy & Auth  | Traefik, Authelia (SSO/2FA), CrowdSec (IDS/IPS)                                  |
| Acceso remoto         | WireGuard (`10.13.13.0/24`), Tailscale                                            |
| Almacenamiento        | Nextcloud (×2 instancias), Immich (fotos), CouchDB (Obsidian LiveSync)            |
| Automatización        | n8n, Guacamole, Portainer                                                         |
| Observabilidad        | Grafana, Prometheus, Splunk (HEC + Traefik JSON logs + CrowdSec + Proxmox syslog)|
| DNS & Conectividad    | ddclient (DDNS), Cloudflare (sin proxy — compatibilidad Nextcloud large files)    |
| CI/CD & Versionado    | Forgejo (sustituyendo Watchtower), Renovate (`datasource=docker`), pre-commit     |

**Aspectos de ingeniería destacados:**

- **Hardening por imagen:** `cap_drop: ALL` con `cap_add` selectivo, análisis de compatibilidad con s6-overlay y healthchecks personalizados para Portainer y Obsidian.
- **Version-pinning:** anotaciones `renovate: datasource=docker` en cada imagen; PRs automáticos de actualización.
- **GitOps:** branch protection en `master`, commits atómicos por servicio, hooks pre-commit para validación de `docker-compose`.
- **Seguridad perimetral:** auditoría de middlewares Authelia — identificación y corrección de servicios expuestos sin SSO (n8n, Grafana, Kasm).
- **Observabilidad:** pipeline Splunk con ingesta desde CrowdSec via HEC, logs JSON de Traefik, syslog de Proxmox y FritzBox.
